The simplest way to use a Gravatar as your website's logo within the WordPress Customizer.


== Description ==

WP Gravatar Logo is the easiest way to use a Gravatar as your website's logo within the WordPress Customizer. Check out the [live demo](https://richtabor.com/ "Live demo of the WP Gravatar Logo") of the plugin in action. Learn more at [ThemeBeans.com](https://themebeans.com/plugins/wp-gravatar-logo?utm_source=home&utm_medium=description_tab&utm_content=home&utm_campaign=readme).


== Installation ==

1. Upload the `wp-gravatar-logo` folder to your `/wp-content/plugins/` directory or alternatively upload the wp-gravatar-logo.zip file via the plugin page of WordPress by clicking 'Add New' and selecting the zip from your local computer.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Customize the size and email for your Gravatar logo via the Customizer > Site Identity section.
